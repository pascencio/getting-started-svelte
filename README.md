# Getting Started Svelte

## Requirements

| Software              | Repository | Optional |
| --------------------- | ---------- | -------- |
| NVM                   |            | no       |
| NPX                   |            | no       |
| Visual Studio Code    |            | yes      |
| Standard.js Extension |            | yes      |

> Before tu run the application you must execute `nvm use` to set correct **node/npm** versions.
